## Explore

- **Analytics** turns the canonical release data into interactive, filterable charts and a searchable run ledger.
- **Forecast Lab** predicts the next observed experiment day and simulates the next calendar month.
- **System Atlas** documents every script, workflow, validation boundary, and data transition.

## Page architecture

The top tab strip and the Starlight sidebar both read from `web/navigation.mjs`. A future page needs one MDX file and one navigation entry, so new project views do not require rebuilding the shell.
