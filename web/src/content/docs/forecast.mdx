<Aside type="caution" title="Small-sample forecasting">
The current dataset contains relatively few active dates. The pipeline therefore emphasizes rolling-origin backtests, robust models, empirical residual intervals, and an explicit confidence score instead of presenting a single precise number as certainty.
</Aside>
